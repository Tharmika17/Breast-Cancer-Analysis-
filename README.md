# Breast Cancer Patient Analysis Dashboard

## 📊 Project Overview

This project analyses breast cancer patient data to explore demographic patterns, tumor characteristics, and survival outcomes.

Using **Python for exploratory data analysis (EDA)** and **Power BI for interactive visualization**, the project transforms raw clinical data into meaningful insights.

The dashboard enables users to quickly explore **patient demographics, cancer stage characteristics, and survival trends**, supporting **data-driven healthcare analysis**.

---

## 🎯 Business Problem

Breast cancer is one of the most common cancers globally. Healthcare professionals and analysts require effective tools to understand how **patient characteristics, tumor properties, and cancer stages influence survival outcomes**.

However, clinical datasets are often complex and difficult to interpret. This project addresses this challenge by developing an **interactive analytical dashboard** that simplifies the exploration of patient data and highlights key patterns related to disease progression.

---

## 🗂 Dataset

The dataset contains clinical records around **4,000 breast cancer patients** with 16 clinical and demographic features.

📄 **Dataset File:**  `Breast_Cancer.csv`

### Key Features

- Age  
- Survival Status (Alive / Dead)  
- Tumor Size  
- Cancer Stages(N, T and 6th)  
- Tumor Differentiation  
- Hormone Receptor Status (Eastrogen and Progestron)
- Regional Lymph Node Examination
- Regional Node Possitive
- Survival Years
- Patient Demographics

## 📈 Dashboard Features

### 👥 Patient Demographics & Survival Overview

This dashboard provides an **important overview of patient demographics and survival outcomes**.

📄 **Power BI Dashboard File:**  `breast_cancer.pbix`

#### Key Metrics
- Total Patients: **4K**  
- Average Survival Year: **6**  
- Average Age: **54**  
- Mortality Rate: **0.15**  
- Average Tumor Size: **30.41 mm**

#### Visual Analysis

- Patient survival status  
- Age group distribution  
- Racial demographics  
- Marital status distribution  
- Cancer grade distribution  

---

### 🧬 Cancer Stage Characteristics

This dashboard focuses on **tumor characteristics and disease progression across cancer stages**.

#### Visual Analysis

- Total Death across Cancer Stages  
- Tumor Differentiation across Cancer Stages  
- Average Tumor size by Cancer Stages  
- Hormone Receptor Status by Cancer Stages  
- Regional Lymph Node Status by Cancer Stages 

#### Key Insights
- **Age Distribution:** Around **2,300 patients (~57%)** fall within the **40–60 age group**, indicating higher prevalence among middle-aged individuals.

- **Survival Outcomes:** Around **3,400 patients (~85%) are alive**, while **~600 patients (~15%) are deceased**, suggesting relatively favourable survival outcomes in the dataset.

- **Tumor Size by Stage:** The **average tumor size is ~30 mm**, with **larger tumors observed in advanced cancer stages**, indicating disease progression.

- **Hormone Receptor Status:** A substantial proportion of patients show **positive hormone receptor status**, with variations across cancer stages.

- **Lymph Node Involvement:** **Advanced stages show higher lymph node involvement**, indicating greater disease spread.
